# PromptCraft - Coding Prompt Generator

PromptCraft is a web application that generates custom coding prompts based on language, task, and additional details. Use this tool to inspire your coding practice and creativity.

## Features

- Generate custom coding prompts based on language, task, and additional details.
- Copy generated prompts to the clipboard.
- Clear input fields for a new prompt.

## Usage

1. Select the coding category (Website, Function, Database).
2. Choose the programming language.
3. Describe the coding task and provide additional details.
4. Click the "Generate" button to get a custom coding prompt.
5. Use the "Copy" button to copy the prompt to the clipboard.
6. Clear fields for a new prompt using the "Clear" button.

## How it Works

PromptCraft uses JavaScript to dynamically generate coding prompts based on user inputs. It provides templates for different coding categories (Website, Function, Database) and combines them with user-supplied details to create unique prompts.
